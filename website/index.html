Welcome

I'm Kennedy, maintainer of the Indie Python project. Here's the apps/games we've published/aim to publish:

Nodezator desktop app published

The Nodezator desktop app is a multi-purpose visual node editor for the Python programming language. It is a subproject of the Indie Python project, but has dedicated channels/content because of its many applications as a tool.

website github video

Dots, Assemble! - pixel art & animation editor not published

A pixel art & animation editor for game sprites. Currently being used to animate characters and objects for the Blue Shooter Man game.

Blue Shooter Man game not published

The first full-fledged game to be released by the Indie Python project. It is a work-in-progress and should have its first beta version released by the end of December 2022 and the full game at the first trimester of 2023.

Dani to the Rescue game prototype published

Made when I was learning pygame. It is not maintained and was published for educational purposes only. Design and overall quality of code is poor, since I was a beginner at the time. There are useful subsystems, though.

github video

Next level: level editor not published

In order to make the Dani to the Rescue game prototype, I needed a way to easily create and edit levels, so this app was made in parallel. It is not ready for release yet.

video

Pynimation: multi-sprite animation editor not published

This is the animation editor I used to help me create the animations used in the Dani to the Rescue game prototype. It is not released yet.
